# Introduction

This project for AI/ML model deployment on cloud using Fastapi Rest endpoints.

## Install dependencies

```
conda install python
pip install fastapi
pip install uvicorn
```

## Run fast api

```
uvicorn app:app --reload
```

## Open Api implementation

```
http://127.0.0.1:8000/docs
```
